## Licensing Overview

TcMenu uses multiple licences depending on the component.

## Why the licensing changed

tcMenu has grown from a single library into a full platform with a modern Web Designer, code generators, plugins, and a stable C++ runtime. The core library remains fully open under Apache 2.0.
The Web Designer and certain advanced plugins now use a protected licence to ensure they remain free for makers and hobbyists while preventing commercial reselling, SaaS hosting, or embedding into proprietary tools without permission.
This model keeps tcMenu open, sustainable, and community‑friendly while protecting the ecosystem from commercial misuse.
