## Archive of Desktop designer 4.4.0

This directory hosts the archived version of tcMenu Designer and plugins
exactly as they existed at V4.4.0. 

We'll not be releasing any further updates to desktop designer, but you
can build it, or use the nightly builds. It should be workable as it was
for at least a few years.

We don't really support it going forward, but you are free to build and 
use this version as long as needed.
